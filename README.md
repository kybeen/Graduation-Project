# 냉장고 관리 서비스 어플리케이션

## 프로젝트 배경 및 동기
- 코로나19로 인해 다양한 식재료 배송 서비스의 활성화 
- 식재료의 다양화와 식재료 구매주문 횟수가 증가되어 통합적인 관리의 필요성이 증가됨

## 프로젝트 목적
- 냉장고 속의 식재료를 핸드폰 어플리케이션으로 어디서든 손쉽게 관리할 수 있도록 하고자 함
